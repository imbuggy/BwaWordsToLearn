# BWA Words to Learn

A polished, high-frequency word learning application designed for Reception and Year 1 students. This tool follows the "Words to Learn" list used by Belleville Wix Academy.

## 🌐 Live Application

You can access the live application here:
[https://ais-pre-aaqdvxccpkd7jf6tijwt7o-697204231429.europe-west2.run.app](https://ais-pre-aaqdvxccpkd7jf6tijwt7o-697204231429.europe-west2.run.app)

## 🚀 Features

- **British Accent TTS**: Clear, natural British English pronunciation.
- **Learning Modes**: Interactive **Read** and **Write** modes.
- **Progress Tracking**: Mastery stats for every word, saved locally.
- **Sync & Backup**: Move your progress between devices with a simple code.
